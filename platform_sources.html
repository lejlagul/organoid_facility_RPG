Organoid Quest: public feature sources

This file documents the public sources used to create the educational platform-choice scoring in the game. The scoring is simplified for public engagement and should not be used as a procurement or experimental design tool.

Last checked: 2026-06-04.

Emulate — Colon Intestine-Chip

Features used in game: biopsy-derived colonoids, colonic endothelial cells, mechanical forces, barrier function, inflammation and drug-response modelling.

https://emulatebio.com/colon-intestine-chip/

https://emulatebio.com/microbiome-application/

CN-BIO — PhysioMimix Gut/Liver-on-a-chip

Features used in game: gut-liver linked model, drug absorption through intestinal barrier, hepatic metabolism, oral bioavailability, organ-to-organ crosstalk and repeated sampling.

https://cn-bio.com/organ-models/gut-liver-on-a-chip/

https://cn-bio.com/organ-models/

BiomimX — uGut / uBeat

Features used in game: uBeat mechanical stimulation resembling intestinal peristalsis, 3D epithelial/endothelial barriers, faecal microbiome and patient-specific microbiome/immunotherapy-response context.

https://www.biomimx.com/ugut/

https://www.biomimx.com/

MEPSGEN — MEPS-X / ProMEPS

Features used in game: automated high-speed/high-precision/high-throughput MPS modelling, reproducibility, customization, multiple tissue/organ model workflows.

https://mepsgen.com/technology/mps.php

https://mepsgen.com/product/promeps.php

HuMiX — human-microbial crosstalk

Features used in game: three microchannels for host cells and microorganisms, porous membranes, physiologically relevant oxygen gradient and sample collection for omics/cytokine/enzyme analyses.

https://www.micalis.fr/equipe/gut-on-chip/

https://www.nature.com/articles/ncomms11535

Dynamic42 — Intestine-on-Chip

Features used in game: gut and vascular compartments, endothelial lining, tissue-resident macrophages, migrated immune cells, flow/biomechanical stimulation, stable operation up to 14 days and barrier/immune/microbiome applications.

https://dynamic42.com/intestine-on-chip/

Mimetas — OrganoPlate

Features used in game: membrane-free OrganoPlate platform, 40–96 chips in microplate format, continuous perfusion, scalable throughput, imaging, TEER, epithelial/organoid, immune-inclusive and mucus models.

https://www.mimetas.com/technology

https://www.mimetas.com/gut
